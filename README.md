# 📊 E-commerce Promotion Analysis

## 📌 Project Overview
This project analyzes the impact of promotional strategies on sales performance using SQL and Power BI. The objective is to identify which promotions drive revenue and improve business decision-making.

---

## 🛠 Tools & Technologies
- SQL (Data Analysis)
- Power BI (Visualization)
- Excel (Data Handling)

---

## 📂 Dataset
- Sales_data_new.csv  
Contains sales, profit, discount, and promotion-related data.

## 📈 Key Insights
- Promotions increased overall sales performance
- Discount-based campaigns showed higher customer engagement
- Identified top-performing products and stores

---

## 💡 Business Impact
- Helps businesses choose effective promotion strategies  
- Improves decision-making using real data  
- Identifies high-performing products and regions
